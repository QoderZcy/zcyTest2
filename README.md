# zcyTest2 - 首页导航系统

一个现代化的首页导航系统，具有流畅的交互体验和响应式设计。

## 功能特点

- ✨ 现代化的导航栏设计
- 📱 完全响应式，支持移动端
- 🎯 单页应用（SPA）体验
- 🎨 优雅的渐变色和动画效果
- ⚡ 流畅的页面切换
- 🔍 支持浏览器前进后退
- ⌨️ 键盘导航支持

## 页面结构

- **首页** - 展示网站特色和功能
- **关于我们** - 公司介绍和团队信息
- **服务** - 提供的服务项目
- **产品** - 产品展示
- **联系我们** - 联系方式和留言表单

## 如何使用

### 方法1：直接打开
在浏览器中直接打开 `index.html` 文件即可。

### 方法2：使用本地服务器
如果您的系统安装了Python，可以运行：

```bash
# Python 3
python3 -m http.server 8080

# 或 Python 2
python -m SimpleHTTPServer 8080
```

如果安装了Node.js，可以使用：

```bash
npx http-server -p 8080
```

然后在浏览器中访问 `http://localhost:8080`

## 文件说明

- `index.html` - 主HTML文件，包含所有页面内容
- `styles.css` - 样式文件，包含所有CSS样式和响应式设计
- `script.js` - JavaScript文件，实现导航逻辑和交互功能

## 技术栈

- HTML5
- CSS3（包含Grid、Flexbox、动画等现代特性）
- 原生JavaScript（无依赖）

## 浏览器支持

- Chrome (推荐)
- Firefox
- Safari
- Edge
- 其他现代浏览器

## 自定义

您可以轻松自定义：

1. **颜色主题** - 在 `styles.css` 中修改渐变色值
2. **导航项** - 在 `index.html` 中添加或删除导航链接
3. **页面内容** - 在 `index.html` 中修改各个section的内容
4. **交互效果** - 在 `script.js` 中自定义JavaScript逻辑

## 特性说明

### 响应式设计
- 桌面端：横向导航栏
- 移动端：汉堡菜单（点击展开/收起）

### 导航功能
- 平滑页面切换
- 活动状态高亮
- 支持URL hash导航
- 浏览器历史记录支持

### 动画效果
- 页面淡入动画
- 导航项悬停效果
- 卡片悬停上浮效果
- 按钮交互动画
